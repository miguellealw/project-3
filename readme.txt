Language used: JavaScript (NodeJS)

IDE: None

Requires NodeJS installed.

Packages used:
    - dgram for udp sockets.
    - fs to read .config
Both packages come built-in with NodeJS and should not require additional setup

Running instructions: Simply run the following command on a terminal. Replace <port> with the port number of each router.
$ node router.js <port>

Example:
$ node router.js 3030
for port number 3030
